How To Read It

  • Per-core bars are system-wide. They include your browser, the FastAPI server, and the OS, not only one worker.
  • One busy Python thread usually means about one logical CPU of total work, even when many cores are allowed.
  • Use Current CPU, PID CPU, and affinity together to reason about migration.

Reading Tip

If a single HIGH worker shows around 100% CPU but the hotspot moves between cores, that usually means the scheduler is migrating one hot thread between eligible CPUs.
